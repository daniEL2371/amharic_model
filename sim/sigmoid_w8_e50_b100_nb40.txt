ግብና : ገቡ መፍረስም ጨፍልቋል አይደሉም ይነገራል
አይኖራቸውም : በክርክር ተፈጻሚነቱም ባኖረ በረጅም ያለውን
ባይባል : ለሚዛናዊነት እስከ ምሥራቅ መሆኑ ምርመራ
በዕውቀት : በዛሬው ክፍለ አምባገነናዊ ገለልተኝነትንና እዚህ
በዲሞክራሲያዊነት : ትኩረት ቃል የማይችል ተደርጎ መርሆ
ጥሩውን : ሊዘነጉ የተቋሙ በኋላ የሚታበል እንደቆሙ
ድክመትን : ባኖረ ትችቶች ማጋነን ሲያበቃ የሚያፍንና
ይታረም : ማስታወቂያ ድህነት ንድፈ እዚህ *
ገደለ : አስቆጥረዋል ይቸገራሉ ያዳግታል አይደሉም አሉ
እንግዲህ : በእኛም የሮማው ከልማታዊ ለእውነትና መገናኛ
እነዚህና : ፀሐፊውን ጉምሩክ ፕሮፓጋንዳ በፖለቲካ አገር
ሌሎች : የአፈና ሥልጣንን ሚዛናዊነትን ገለልተኝነትንና ግን
በርካታ : በእነዚሁ የመውሰድ በደርግ በንጉሡ ዋናው
የመንግሥትና : በመሆናቸው ፖሊስ በልማቱ ተጭኗቸው በሥልጣን
ጫፍና : ኢንዱስትሪ ከፈላጭ የማለቱን አይደሉም መዋቅሮችን
የረገጡ : እንዲነሳ የገቢዎችና ሰጪ የሚያስችሉ ዜጎችን
ድክመቶችን : ኅትመት መጽሔቶች ፈቅደው ነፃነትና አገር
በመገንዘቤ : ቫን ተጫውቷል የለበትም ይታያል ተቀላቅለዋል
የተገነባ : አትሞ መሀል እንዳይኖር ንጉሣዊ እንዳልዘነጋሁ
መከራከሪያ : ንዑስ ስለሆነ አስሯል አገዛዝ እንዳልዘነጋሁ
የዘርፉን : ለማጠቃለል የአዲስ ዋነኛ ዋናው ቀዳሚው
እየቆጠቆጠ : ብሔራዊነት ሁለተኛው ይታዩና ኃላፊዎች የብሮድካስት
ከመጣው : ምርምር ታስረን መራሹ ዕገዳና ግን
ፍልስፍናና : ይሉታል ማግኘት የምዕራቡ የሚያስችሉ እየተስፋፋ
ምኞት : ልማድ ፖሊሲው በሙሉ ባህሪ ቆራጭ
መመልከትን : ትምህርት መሸፋፈን እንዴት ከመሆን እንዳይኖር
መርጫለሁ : አይድንም ሎክ ጽፈዋል አሳድዷል አይደሉም
የሚስማማ : ሥልጣንን ሐሳብ ገለልተኝነትንና ግን *
ካለ : የማይመረምር ገብነት ፈቅደው መለኮታዊነት ገለልተኝነትንና
ብንወያይበትም : ብል ቀጣይነት የሚጣጣሩትን ለመጠበቅ እዚህ
ይበጅ : እንዳለበት በብዛት ሊኖረው አይደለንም መሆኑን
