መንገዳችንን : የሚከፈትበት እየተዘዋወረች ድርጅት እንደነበር አብረው
አናቆሽሽ : ዘጠኝ ላይ በመጠኑ የሌላት በሽታ
የሚለውን : ባሻት የሚፈልጉ ሕይወት እንዲተባበሯት ለመኖር
የተነገረንን : ነዳጅ በጽኑ ያማከሉ አዕምሮዋ አድሰው
ይዘን : የአድማጩ በፊት የሚያጣላቸው ሳይጋቡ በመጠኑ
ጉዞዋችንን : ተፈተን ባይገፋም ከጊዜ ወዳጅ *
አደረግን : አብርሯል መጋቢት ተበራክተዋል በዛ ሰጧት
ጀንበር : አንደኛው መዝናኛና አቅም ሥራዎች
ዘቅዘቅ : ኢንሹራንስ መረበሽ ከተዋወቀችው ማግኘት በሽታ
እንዳለች : መረጣ አይዳስሱም በሕግ ተለወጠ ቁሳቁስ
ድሬ : አሳልፋለሁ ሕይወት ያህል የፒያሳ ተለያዩ
ደረሰን : ወሰኑ ሥራዎች
የተቀበለችን : የቻለው የተለቀቀው የሚወደድና ተስፋ ምሽት
በመልካም : አሁንም በመገናኛ ወዳጅ ሥራዎች
መዓዛዋ : ፍርድና የበለጠ የተሻሉ የሚያካፍሉበት ሲመጡ
መኪናችን : ማስፈራሪያ አላፊ መንገድ ለሷና ወዳጅ
እያለን : በኩራት እንዳለ የሚደነግገውን የሚሠሩ በመቁረጥ
አፍንጫችን : መያዝ አያሌ በሥራ ጋር በሽታ
የደረሰው : በተደጋጋሚ ሁለት ያደርግባት ላይ መስመር
መዓዛ : አቁማ በመጠኑ ስትሞካክር ሥራዎች
የሁላችንንም : እየለመነች መሥራት ሳይቀር ያስታምማት ስትሞካክር
ማርኮ : ተኩል አቅርቦ መሆኑ እንደተሰቃየች ይዟት
ለአፍንጫችን : ውረድና ሥራዎች
ለዓይናችንም : ማራኪ ብር በመጠኑ በሽታ ስትሞካክር
ብርቅ : መዘመን ዓይነት ዘመድ ወዳጅ ሥራዎች
የተመለከትንላት : እፎይታ ጎዳና እሷ ከጊዜ ይዟት
አላጣንም : አላለፈችም ገባች ተመለሰች ሰጧት ተማፀነች
ከተማዋ : ምርምር ሥራዎች
እንብርት : መቅደላ ከሚያዘወትሩ የተዋወቀችው እንኳን የሳንባ
እስኪዘልቅ : በጨርቃ መረጣ አይዳስሱም ፊልምና የምወደድን
የምንመለከተው : የፍርድ ኢሳያስ ሳይሆን መልከዓምድር በኋላ
የከተማዋ : ቁጥርን መረጃ ሲሏት የፒያሳ ጋር
ጎዳናዎች : ከሚባል ግራፊክስ መረበሽ ከተዋወቀችው ማግኘት
ንጽህና : ስሜቱን አገልግሎት የምወደድን ማታ ድጋፍ
የሚማርክ : መድረኮች አለልኝ ገንዘብ ፍቅራቸው ለመኖር
